# Badminton Split-Bill

Project kecil ini dibuat buat satu hal: bikin urusan bayar badminton jadi lebih gampang dan transparan.

Semua orang bisa lihat estimasi, pembagian biaya per orang, dan status pembayaran tanpa perlu tanya-tanya manual di chat.

Intinya biar main tetap fun, urusan split bill tetap jelas.
